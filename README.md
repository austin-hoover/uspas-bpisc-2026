# USPAS BPISC 2026

Lecture notes for USPAS course *Beam Physics with Intense Space Charge (BPISC)*, July 2026.


## Environment

Install Python dependencies in [conda](https://conda-forge.org/download/) environment:

```
conda env create --file environment.yaml
conda activate uspas-bpisc-2026
```

Install [quarto](https://quarto.org/docs/get-started/).


## Build

Run `make` to run scripts, render slides, and move output to new folder.
